Anlisis Predictivo - Prediccin del PIB

Introduccin

En este anlisis, se busca predecir el Producto Interno Bruto (PIB) de diferentes pases en funcin de diversas caractersticas econmicas y demogrficas. Las variables seleccionadas incluyen el PIB per cpita, poblacin, esperanza de vida, emisiones de CO2, y el porcentaje de exportaciones e importaciones respecto al PIB.

El anlisis se enfoca en comparar dos modelos predictivos:

  1. Regresin Lineal: Un modelo bsico de prediccin que busca una relacin lineal entre las variables.
  2. Random Forest: Un modelo avanzado de machine learning basado en rboles de decisin, ideal para capturar relaciones no lineales.

Ambos modelos sern evaluados usando mtricas como RMSE (Error Cuadrtico Medio), R (Coeficiente de Determinacin) y MAE (Error Absoluto Medio). Estos modelos permitirn identificar cul tiene un mejor desempeo en la prediccin del PIB de los pases a partir de las caractersticas seleccionadas.

Objetivos y Mtodos

Objetivo

El objetivo de este anlisis es construir y evaluar modelos predictivos para predecir el PIB de los pases en funcin de variables econmicas y demogrficas. A travs de dos modelos:

  1. Regresin Lineal: Para encontrar una relacin lineal entre las variables.
  2. Random Forest: Para capturar relaciones no lineales y manejar la complejidad de las interacciones entre variables.

Mtodos

  1. Preparacin de los Datos:

    • Eliminar valores nulos.
    • Normalizar las variables para mejorar la precisin de los modelos predictivos.

  2. Modelado Predictivo:

    • Regresin Lineal: Entrenamiento de un modelo sencillo de regresin.
    • Random Forest: Entrenamiento de un modelo basado en mltiples rboles de decisin.

  3. Evaluacin de los Modelos:

    • Uso de RMSE, R y MAE para comparar el desempeo de ambos modelos.
    • Visualizacin de las predicciones versus los valores reales para cada modelo.

Preparacin de los Datos

Limpieza de Datos

Es esencial limpiar los datos antes de entrenar los modelos. Esto incluye:

  1. Eliminacin de Valores Nulos: Se eliminaron las filas con valores faltantes en las variables seleccionadas.
  2. Normalizacin/Estandarizacin: Algunas variables, como el PIB o las emisiones de CO2, pueden tener rangos muy amplios. Por lo tanto, se estandarizaron las caractersticas para asegurarse de que todas las variables tengan el mismo peso en el modelo, especialmente para el modelo de Regresin Lineal.
  3. Variables Seleccionadas: Se utilizaron las siguientes variables:
    • GDP_per_capita_USD: PIB per cpita.
    • Population: Poblacin total.
    • Life_exp: Esperanza de vida.
    • CO2_emissions: Emisiones de CO2.
    • Exports_percent_GDP: Porcentaje de exportaciones respecto al PIB.
    • Imports_percent_GDP: Porcentaje de importaciones respecto al PIB.

Estas caractersticas se usaron para predecir el PIB (variable dependiente).

import pandas as pd
from sklearn.model_selection import train_test_split
from sklearn.preprocessing import StandardScaler

# Cargar el dataset
file_path = './output/WDI_Complete_Analysis_PowerBI.csv'
dataset = pd.read_csv(file_path)

# Variables y objetivo
features = ['GDP_per_capita_USD', 'Population', 'Life_exp', 'CO2_emissions', 'Exports_percent_GDP', 'Imports_percent_GDP']
target = 'GDP'

# Filtrar los datos y eliminar valores nulos
dataset_cleaned = dataset[features + [target]].dropna()

# Separar las variables predictoras (X) y la variable dependiente (y)
X = dataset_cleaned[features]
y = dataset_cleaned[target]

# Normalizar las caractersticas
scaler = StandardScaler()
X_scaled = scaler.fit_transform(X)

# Dividir los datos en conjunto de entrenamiento (80%) y conjunto de prueba (20%)
from sklearn.model_selection import train_test_split
X_train, X_test, y_train, y_test = train_test_split(X_scaled, y, test_size=0.2, random_state=42)

# Verificar dimensiones de los datos
X_train.shape, X_test.shape, y_train.shape, y_test.shape

Seleccin del Modelo Predictivo

1. Regresin Lineal

La Regresin Lineal es un modelo predictivo que asume que existe una relacin lineal entre las variables independientes y la variable dependiente (en este caso, el PIB). Este modelo es ideal cuando las relaciones entre las variables son simples y lineales.

2. Random Forest

El Random Forest es un modelo de machine learning ms avanzado que utiliza mltiples rboles de decisin para predecir el resultado. Este modelo es ms flexible y puede manejar relaciones no lineales entre las variables, lo que lo hace adecuado para datos ms complejos.

In[2]:
from sklearn.linear_model import LinearRegression
from sklearn.ensemble import RandomForestRegressor
from sklearn.metrics import mean_squared_error, r2_score, mean_absolute_error

# Entrenar el modelo de Regresin Lineal
linear_model = LinearRegression()
linear_model.fit(X_train, y_train)
y_pred_linear = linear_model.predict(X_test)

# Evaluar el modelo de Regresin Lineal
mse_linear = mean_squared_error(y_test, y_pred_linear)
rmse_linear = mse_linear ** 0.5  # Calcular RMSE manualmente
r2_linear = r2_score(y_test, y_pred_linear)
mae_linear = mean_absolute_error(y_test, y_pred_linear)

# Entrenar el modelo de Random Forest
random_forest_model = RandomForestRegressor(n_estimators=100, random_state=42)
random_forest_model.fit(X_train, y_train)
y_pred_rf = random_forest_model.predict(X_test)

# Evaluar el modelo Random Forest
mse_rf = mean_squared_error(y_test, y_pred_rf)
rmse_rf = mse_rf ** 0.5  # Calcular RMSE manualmente
r2_rf = r2_score(y_test, y_pred_rf)
mae_rf = mean_absolute_error(y_test, y_pred_rf)

# Resultados de evaluacin
evaluation_results = {
    'Modelo': ['Regresin Lineal', 'Random Forest'],
    'RMSE': [rmse_linear, rmse_rf],
    'R': [r2_linear, r2_rf],
    'MAE': [mae_linear, mae_rf]
}

evaluation_df = pd.DataFrame(evaluation_results)

# Mostrar las mtricas
evaluation_df
